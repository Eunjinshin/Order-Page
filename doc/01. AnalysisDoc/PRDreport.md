## 1. 기능 요구사항

### 1.1 회원·인증

| ID | 기능명 | 설명 |
|----|--------|------|
| F-001 | 이메일 회원가입 | 이메일·비밀번호·이름·연락처 입력으로 계정 생성, 인증 메일 발송 |
| F-002 | 비밀번호 유효성 검증 | 최소 8자, 영문+숫자+특수문자 조합, 실시간 강도 표시 |
| F-003 | 이메일/비밀번호 로그인 | 이메일·비밀번호 로그인, 5회 실패 시 30분 계정 잠금 |
| F-004 | 구글 OAuth 2.0 로그인 | Google 소셜 로그인, 신규 사용자는 자동 회원가입 처리 |
| F-005 | JWT 토큰 인증 | Access Token(1시간) + Refresh Token(30일) 발급 및 자동 갱신 |
| F-006 | 비밀번호 찾기/재설정 | 이메일로 재설정 링크 발송, 링크 유효 시간 24시간 |
| F-007 | 회원 정보 수정 | 이름·연락처·배송지 주소 수정, 이메일 변경 시 재인증 |
| F-008 | 로그아웃 / 회원 탈퇴 | 로그아웃 시 토큰 폐기, 탈퇴 시 개인정보 30일 후 삭제 |
